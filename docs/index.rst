sphinx-riddle-whisper
=====================

Sphinx 拡張 ``sphinx-riddle-whisper`` のドキュメント兼デモ。

``:term:`` 参照に対して、glossary（用語集）の定義本体を切り詰めずに
ホバー / クリックでポップアップ表示する拡張です。

.. note::

   本ドキュメントは開発中です。デモ用の用語集ページは今後追加されます。

ネストポップアップのデモ
========================

:term:`popover` のポップアップ内にある :term:`glossary` などの ``:term:``
リンクにホバーすると、2段目のポップアップが重ねて表示されます（固定2段）。
クリックした場合は、その用語の定義元（用語集）が新しいタブで開きます。
